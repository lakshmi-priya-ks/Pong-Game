# 🏓 Pong Game – 20 Games Challenge (1/20)

This is a classic Pong game developed using Unity as part of the 20 Games Making Challenge.

---

## 🎮 Features

- Two-player paddle controls
- Ball movement and bouncing
- Collision with walls and paddles
- Score tracking system
- Ball resets after scoring

---

## 🛠️ Engine Used

Unity (2D Core)

---

## 🎯 Controls

| Player | Keys |
|--------|------|
| Left Paddle | Up Arrow / Down Arrow |
| Right Paddle | I / K |
| Both Paddles (Simultaneous) | W / S |
